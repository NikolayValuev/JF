1. Git - сис-ма контроля версий
2. GitHub - веб сервис для хранения проектов и их обработки, он основан на Git
3. Ячейка в которой хранятся данные на GitHub - РЕПОЗИТОРИЙ (REPOSITORY). У каждого репозитория на GitHub есть свой уникальный адрес в сети - URL пример:(https://github.com/NikolayValuev/JavaRush_tasks)
Сама по себе ссылка на репозиторий состоит из:
https://github.com/ - общий домен сервера
NikolayValuev/ - имя пользователя в GitHub, он владелец репозитория
JavaRush_tasks - имя самого репозитория
