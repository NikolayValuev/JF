1.Элементы графического интерфейса ToolWindow (см. scrin T/W)
  они разбиты на 3-и группы: 
  левый край основного окна/ правый / нижний край осн.окна
 
2.  Project ToolWindow:
    используется для редактирования файлов проекта, внося здесь            изменения эти изменения отображаются в файловой системе компьютера. (см.Screen изменения файлов)

3. Редактор Editor:
   все файлы редактируются в редакторе, он предоставляет такие фичи как подсветка текста / автодополнение текста / форматирование текста. Чтобы отредактировать с его помощью файл, необходимо выбрать его в ProjectToolWindow двойным щелчком левой кл.мыши
